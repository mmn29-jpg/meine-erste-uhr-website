GitHub

Eine einfache Übersicht für deine erste eigene Website.

GitHub

  • Plattform zum Speichern von Code (Repositories).
  • Für Programmierer und Teams.
  • Versionsverwaltung mit Git.
  • Man kann Projekte entwickeln und gemeinsam bearbeiten.

GitHub Pages

  • Dienst von GitHub zum Veröffentlichen von Webseiten.
  • Nimmt Dateien aus einem GitHub-Repository und macht daraus eine Website.
  • Kostenlos für statische Webseiten (HTML, CSS, JavaScript).
  • Deine Webseite bekommt z. B. die Adresse:
https://dein-name.github.io/dein-projekt/
